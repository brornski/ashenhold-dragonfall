# User-supplied biome skybox provenance

## Ember Dunes sandskybox

Runtime file: `ember-dunes-sandsky-2k.png`

The project owner supplied `sandskybox.zip` on July 20, 2026 and explicitly requested that its desert panorama be incorporated into Ashenhold: Dragonfall. The archive contained `sky_12_2k.png` plus six equivalent cubemap faces, but did not contain original author, source URL, copyright, or license metadata.

Ashenhold ships the single 2048 x 1024 equirectangular panorama under the project owner's authorization. The image is renamed for its runtime role and is otherwise unmodified. If upstream authorship or licensing information becomes available, update this notice and `ATTRIBUTIONS.md` before the next release.

## Moonfall moonsky

Runtime file: `moonfall-moonsky-2k.png`

The project owner supplied `moon skybox.zip` on July 20, 2026 and explicitly requested that its panorama replace the Moonfall biome sky in Ashenhold: Dragonfall. The archive contained `sky_17_2k.png`, six equivalent cubemap faces, and a cubemap-layout reference image, but did not contain original author, source URL, copyright, or license metadata.

Ashenhold ships the single 2048 x 1024 equirectangular panorama under the project owner's authorization. The image is renamed for its runtime role and is otherwise unmodified; the redundant cubemap faces and layout reference are not shipped. If upstream authorship or licensing information becomes available, update this notice and `ATTRIBUTIONS.md` before the next release.
