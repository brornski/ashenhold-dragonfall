# Ember Dunes sandskybox provenance

Runtime file: `ember-dunes-sandsky-2k.png`

The project owner supplied `sandskybox.zip` on July 20, 2026 and explicitly requested that its desert panorama be incorporated into Ashenhold: Dragonfall. The archive contained `sky_12_2k.png` plus six equivalent cubemap faces, but did not contain original author, source URL, copyright, or license metadata.

Ashenhold ships the single 2048 x 1024 equirectangular panorama under the project owner's authorization. The image is renamed for its runtime role and is otherwise unmodified. If upstream authorship or licensing information becomes available, update this notice and `ATTRIBUTIONS.md` before the next release.
