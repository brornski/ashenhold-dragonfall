## Project adaptations

- `F1_Tree1.glb` derives from `F1_Tree1.fbx`.
- `F1_Tree3.glb` derives from `F1_Tree3.fbx`.
- The source FBX files were converted with FBX2glTF 0.9.7 and optimized with glTF-Transform 4.4.1 using weld and quantization transforms.
- The runtime files use `KHR_mesh_quantization`, contain no external textures, and are intended to receive Ashenhold's biome materials at runtime.
- The original 19 MB bark normal map and the remaining foliage-pack textures/models are intentionally not redistributed with this web build.
