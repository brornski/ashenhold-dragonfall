# User-supplied Flags asset notice

Source archive: `Flags1_(AssetPack3).zip`, supplied by the Ashenhold project owner with an explicit request to use its flag models in the game.

The archive did not include original author, storefront, or license metadata. This notice records project provenance and the owner's authorization to include the supplied files; it does not invent or replace any third-party license that may govern the original pack. The project owner is responsible for retaining the original purchase/download records and confirming redistribution rights before these derivatives are distributed outside Ashenhold.

Runtime adaptations:

- `flag-cloth.glb` derives from `AP3_Flag.fbx` and uses a compact blue Warden texture derived from `AP3_Flag_8.tga`.
- `flag-banner.glb` derives from `AP3_Banner.fbx` and uses a compact hostile texture derived from `AP3_Banner_4.tga`.
- `flag-emblem.glb` derives from `AP3_Emblem.fbx` and uses a compact Warden emblem derived from `AP3_Emblem_8.tga`.
- Source textures were transcoded losslessly to WebP for same-origin browser delivery.

The FBX meshes were converted to binary glTF with the open-source FBX2glTF conversion tool. FBX2glTF is not shipped with the game, and its tool license does not establish the license of the converted artwork.
