# User-supplied Frostbound tree-pack notice

Source archives: `christmas trees.zip` and `winter tree yellow.zip`, supplied by the Ashenhold project owner with an explicit request to blend their trees throughout Frostbound Wilds.

Each archive contained three GLB levels of detail and no original author, storefront, or license metadata. This notice records project provenance and the owner's authorization to include the supplied files; it does not invent or replace any third-party license that may govern the original packs. The project owner is responsible for retaining the original purchase/download records and confirming redistribution rights before these derivatives are distributed outside Ashenhold.

Runtime adaptations:

- Geometry from all three supplied LODs is retained for both tree families.
- Repeated embedded textures were externalized once per family; unused normal and packed material maps were omitted.
- Branch color maps were transcoded to compact WebP, while cutout foliage maps use lossless WebP to preserve alpha edges.
- Frostbound placements deterministically alternate between the green Christmas and yellow winter families. No other biome uses these derivatives.
